# 学习画像、学习路径与评价

## 用行为证据更新学习画像

学习画像中的知识基础不能只依赖一次对话或一次 LLM 判断。系统应持续组合练习正确率、错误类型、完成时间、前后测变化、作业诊断和对话证据，形成可复核的行为证据，再校准学生对各知识点的掌握程度。练习反馈既能暴露薄弱点，也能验证此前画像判断是否准确。

## 动态学习路径

个性化学习路径需要随学习过程动态调整。学生的目标、掌握程度和学习节奏发生变化后，系统应根据最新反馈重新安排资源、练习和复习顺序，并保留调整原因，避免把早期判断永久固化。

## 形成性评价与总结性评价

形成性评价贯穿学习过程，强调及时反馈和教学调整，例如随堂练习、阶段诊断与错因分析。总结性评价通常发生在一个学习阶段结束时，用于衡量整体学习结果。两类评价用途不同，不能只用期末结果代替持续反馈，也不能把单次练习当作最终能力结论。
